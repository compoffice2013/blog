# Rank Math SEO 100点獲得ガイドライン

Rank Mathで100点満点を獲得するための要件一覧です。ブログ記事作成時にこのリストをチェックしてください。

## 1. 基本SEO設定 (Basic SEO)
- [ ] **SEOタイトル**: フォーカスキーワードを含める（できれば冒頭）。50文字以内推奨。
- [ ] **メタディスクリプション**: フォーカスキーワードを含める。120〜160文字。
- [ ] **URL（パーマリンク）**: フォーカスキーワードを含める。短くシンプルに（75文字以内）。
- [ ] **冒頭のキーワード**: 記事の最初の10%（最初の段落）にフォーカスキーワードを含める。
- [ ] **コンテンツ内のキーワード**: 全体を通して自然にフォーカスキーワードを使用する。
- [ ] **文字数**: 
    - 600語以上: 合格ライン
    - 2500語以上: 100%スコア（長文コンテンツの場合）

## 2. 追加SEO要素 (Additional SEO)
- [ ] **見出し (H2/H3)**: サブ見出しにフォーカスキーワードを含める。
- [ ] **画像 ALT属性**: 画像のALTテキストにフォーカスキーワードを含める。
- [ ] **キーワード出現率**: 1%〜1.5%を目指す（詰め込みすぎない）。
- [ ] **URLの長さ**: 75文字以下。
- [ ] **外部リンク**: 信頼できる外部サイトへのリンクを少なくとも1つ含める（Dofollow）。
- [ ] **内部リンク**: 自サイト内の関連ページへのリンクを含める。
- [ ] **キーワードの独自性**: 過去の記事と同じフォーカスキーワードを使わない。

## 3. タイトルの読みやすさ (Title Readability)
- [ ] **キーワードの位置**: SEOタイトルの**先頭**にフォーカスキーワードを配置する。
- [ ] **感情喚起 (Sentiment)**: ポジティブまたはネガティブな強い感情を引く言葉を入れる。
- [ ] **パワーワード**: クリックしたくなる「パワーワード」を含める（例：「驚愕」「秘密」「最高の」「たった1つの」など）。
- [ ] **数字**: タイトルに数字を含める（例：「3つの理由」「2026年版」など）。

## 4. コンテンツの読みやすさ (Content Readability)
- [ ] **目次 (TOC)**: 目次プラグインが機能するように、適切なH2/H3構造を作る。
- [ ] **短い段落**: 1つの段落は120語（日本語なら3〜4行程度）以内に収める。
- [ ] **メディア (画像/動画)**: 
    - 記事内に画像や動画を含める。
    - **100%スコアには4つ以上の画像/動画が必要**。

## ワークフローへの適用事項
1. **タイトル作成時**: 数字とパワーワードを必ず入れる。
2. **構成時**: H2見出しにキーワードを入れる。
3. **執筆時**: 段落を短く切る。
4. **リンク**: 内部・外部リンクを本文中に自然に入れる。
5. **画像**: 記事納品後、ユーザー側で4枚以上の画像配置を推奨するコメントを入れる。
